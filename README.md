### Commands
* **man** — Print manual or get help for a command
    * Ex.: **man ls** (shown information for command ls)
* **pwd** — Print working directory
* **cd**  — Change directory
    * .. (back)
    * \ (define space) Ex.: cd arquivo\ morto
* **ls** — List directory contents
    * -a (all contents)
    * -l (more information)
